# Workshop Exercises Pack

This pack maps directly to conference delivery and can be used in guided forms, printable worksheets, or facilitator-led group practice.

## Exercise 1: Accessibility Journey Check-In
- Time: 20 minutes
- Purpose: establish participant context and partner barriers.
- Output: journey notes and shared challenge map.

## Exercise 2: What Problem Are We Solving?
- Time: 35 minutes
- Purpose: move from tool-first thinking to problem-first framing.
- Output: problem statement with capacity-building target.

## Exercise 3: Fix It for Me vs Teach Me to Improve It
- Time: 40 minutes
- Purpose: practice coaching language that solves now and teaches next time.
- Output: reusable coaching response examples.

## Exercise 4: Helpful, Risky, or Human Required
- Time: 40 minutes
- Purpose: classify tasks by responsible AI boundary.
- Output: AI boundary map with safeguards.

## Exercise 5: Accessibility Agent Formula
- Time: 45 minutes
- Formula: Role + Task + Trusted Guidance + Output Format + Human Review
- Output: first non-technical accessibility agent concept.

## GLOW Lab 1: Accessible Communications
- Time: 60 minutes
- Focus: plain language, structure, meaningful links, inclusive access language.
- Output: revised communication and review checklist.

## GLOW Lab 2: Alt Text and Human Judgment
- Time: 55 minutes
- Focus: image purpose, relevant details, omission decisions, verification.
- Output: image-description coaching checklist.

## GLOW Lab 3: Remediation Planning
- Time: 55 minutes
- Focus: prioritization, owner coaching, realistic sequencing.
- Output: remediation coaching template.

## Accessibility Champion Studio
- Time: 50 minutes
- Focus: build a reusable partner-facing workflow.
- Output: draft champion workflow artifact.

## Peer Review and Refinement
- Time: 40 minutes
- Focus: strengthen trust, safeguards, and reuse quality.
- Output: revised workflow + scaling path.

## Capstone and Action Plan
- Time: 15 minutes
- Focus: commitment to operational follow-through.
- Output: 30-day plan with one workflow, one partner, one safeguard, one next step.
