What is a template?

A Word template (.dotx) is a starting point for new documents. When you create a new document from this template, all ACB Large Print styles are pre-configured: Arial font, correct sizes for headings and body text, flush-left alignment, proper spacing, and margins. You do not need to format anything manually.

How do I use the template in Microsoft Word once it's downloaded?

The downloaded file is named ACB-Large-Print-Template.dotx. There are three common ways to use it in Word for Windows or Word for Mac.

Option 1 -- Quick: open it once for a single new document

  1. Locate ACB-Large-Print-Template.dotx in your Downloads folder.
  2. Double-click the file. Word opens a brand-new untitled document that already has the ACB styles applied.
  3. Use File → Save As and choose Word Document (.docx) to save your new document. The template stays untouched and reusable.

Tip: do not save back over the .dotx -- treat it as a starter, like a blank form.

Option 2 -- Recommended: install it as a Personal Template (Windows)

  1. In Word, open File → Options → Save.
  2. Note (or set) the Default personal templates location. A common path is C:\Users\<you>\Documents\Custom Office Templates.
  3. Move ACB-Large-Print-Template.dotx into that folder.
  4. From now on, in Word choose File → New → Personal. The ACB Large Print template appears as a tile -- click it to start a new document.

Option 2 -- Recommended: install it as a Personal Template (macOS)

  1. In Finder, press ⇧ + ⌘ + G and go to ~/Library/Group Containers/UBF8T346G9.Office/User Content.localized/Templates.localized/.
  2. Drag ACB-Large-Print-Template.dotx into that folder.
  3. In Word for Mac, choose File → New from Template and pick the Personal tab. The ACB Large Print template appears there.

Option 3 -- Apply the template's styles to an existing document

If you already have a draft and want to copy in just the ACB styles (Arial 18pt body, correct heading sizes, etc.):

  1. Open your existing document in Word.
  2. Go to Developer → Document Template (Windows) or Tools → Templates and Add-ins (Mac). If the Developer tab isn't visible on Windows, enable it under File → Options → Customize Ribbon.
  3. Click Attach..., browse to ACB-Large-Print-Template.dotx, and click Open.
  4. Tick Automatically update document styles, then click OK. Your existing content is restyled to match ACB defaults.

If you don't see the change, the existing document may have direct (manual) formatting that overrides styles. Run GLOW's Fix tool on the result to clear any remaining ACB violations.

Trust prompt and macros

This template contains no macros and no embedded scripts. If Word shows a "Protected View" or "Trust Center" prompt the first time you open it, you can safely click Enable Editing.

What does sample content include?

The sample content adds example headings (H1, H2, H3), body paragraphs, a bulleted list, a numbered list, and a simple table -- all formatted to ACB standards. This helps you see what correctly formatted content looks like. Delete the sample content when you start writing your own document.

When do I need a binding margin?

If your document will be printed and bound (stapled, spiral, or three-ring), add a binding margin. This shifts all content 0.5 inches to the right so that text near the left edge is not hidden by the binding. For digital-only documents or unbound printouts, leave this unchecked.
