Active branding profile

Current profile: {% if brand_is_uarizona %} uarizona — University of Arizona theme (Arizona Red & Navy, UA wordmark logo). {% else %} bits — BITS default theme (navy, BITS logo). This is the default when no profile is set. {% endif %}

The branding profile is a deployment-level setting and cannot be changed here. To switch profiles, set the GLOW_BRAND_PROFILE environment variable before starting the container:

  • GLOW_BRAND_PROFILE=bits — BITS / ACB theme (default)
  • GLOW_BRAND_PROFILE=uarizona — University of Arizona theme

In docker-compose.yml add it under the environment: key, then redeploy. A restart is required for the change to take effect.
