How does heading review work?

The tool analyzed your document and found paragraphs that look like headings (bold, large font, short text) but are not using heading styles. Each candidate is shown with the signals that triggered detection and a confidence score.

For each candidate, you can:

  • Accept the suggested heading level (the default selection)
  • Change the heading level using the dropdown
  • Select "Not a heading -- skip" to leave the paragraph unchanged
