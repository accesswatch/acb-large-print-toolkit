What does convert do?

Two conversion directions are available:

To Markdown (via Microsoft MarkItDown)
Transforms Office documents and PDFs into clean Markdown text. Useful for text analysis, LLM pipelines, or creating accessible Markdown versions. Supports: Word (.docx), Excel (.xlsx), PowerPoint (.pptx), PDF (.pdf), HTML (.html), CSV (.csv), JSON (.json), XML (.xml), ePub (.epub), and ZIP (.zip).
To ACB HTML (via Pandoc)
Converts text documents into standalone, accessible HTML with embedded ACB Large Print CSS. Output is ready for web hosting, email, or CMS embedding. Supports: Markdown (.md), reStructuredText (.rst), OpenDocument (.odt), Rich Text (.rtf), and Word (.docx).

Conversion Direction

To Markdown: Word, Excel, PowerPoint, PDF, HTML, CSV, JSON, XML, ePub, ZIP. To ACB HTML: Markdown, RST, ODT, RTF, Word. Maximum size: 16 MB.

What is the output?
To Markdown
A .md file preserving the document's structure -- headings, lists, tables, and links -- as plain text with minimal markup. Images are not extracted; their alt text or captions are preserved where available.
To ACB HTML
A standalone .html file with embedded ACB Large Print CSS (Arial, 18pt body, 22pt headings, flush left, proper spacing, WCAG 2.2 contrast). Ready to open in any browser or paste into a CMS.
