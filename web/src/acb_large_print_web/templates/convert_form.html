What does convert do?

Convert uses Microsoft MarkItDown to transform your document into clean Markdown text. This is useful for text analysis, LLM pipelines, or creating accessible Markdown versions of Office documents and PDFs.

Supported formats: Word (.docx), Excel (.xlsx), PowerPoint (.pptx), PDF (.pdf), HTML (.html), CSV (.csv), JSON (.json), XML (.xml), ePub (.epub), and ZIP (.zip).

Audio files (.mp3, .wav) are not supported in the web interface for privacy reasons -- audio transcription requires sending data to external APIs.

What is the output?

You will receive a .md (Markdown) file preserving the document's structure -- headings, lists, tables, and links -- as plain text with minimal markup. Images are not extracted; their alt text or captions are preserved where available.
