What does the fix do?

For Word documents, the fixer automatically corrects ACB Large Print compliance issues: fonts to Arial, sizes to 18pt/22pt/20pt, italic removed, bold emphasis converted to underline, flush-left alignment, line spacing, margins, widow/orphan control, hyphenation off, and language set. You download a fixed copy -- your original is not modified.

For Markdown files, the fixer corrects heading hierarchy, italic removal, and em-dash replacement. Other Markdown issues require manual review.

For Excel, PowerPoint, and PDF files, auto-fix is not yet available. You will receive an audit report with detailed findings and manual fix guidance instead.

How does the fix work?

The tool always applies all available auto-fixes to your document to ensure maximum compliance. The mode you select above controls which rules are highlighted in the results report:

  • Full Fix shows all rule results.
  • Essentials Fix shows only Critical and High severity results.
  • Custom Fix shows only the rules you select.

Regardless of mode, your downloaded Word document will have all auto-fixable issues corrected. Some issues (like heading hierarchy and document title) require manual attention.

For heading detection, use the heading accuracy mode to tune behavior: Conservative reduces false positives, Balanced is the default, and Thorough captures more possible headings for review.

For Excel and PowerPoint, the tool runs a full audit and provides a detailed remediation guide. Auto-fix support for these formats is planned for a future release.
