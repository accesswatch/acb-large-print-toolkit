What does the fix do?

The fixer automatically corrects ACB Large Print compliance issues in your Word document. It changes fonts to Arial, adjusts sizes to 18pt/22pt/20pt, removes italic formatting, converts bold emphasis to underline, sets flush-left alignment, corrects line spacing, fixes margins, enables widow/orphan control, disables hyphenation, and sets the document language. You download a fixed copy -- your original file is not modified.

How does the fix work?

The tool always applies all available auto-fixes to your document to ensure maximum compliance. The mode you select above controls which rules are highlighted in the results report:

  • Full Fix shows all rule results.
  • Essentials Fix shows only Critical and High severity results.
  • Custom Fix shows only the rules you select.

Regardless of mode, your downloaded document will have all auto-fixable issues corrected. Some issues (like heading hierarchy and document title) require manual attention.
