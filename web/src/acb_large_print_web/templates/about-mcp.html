GLOW MCP Server API (7.2.0)

New in 7.2.0: GLOW now includes a Model Context Protocol (MCP) server for agent-to-agent integration and automation. The MCP server exposes GLOW’s accessibility audit, fix, convert, and reporting features via a modern API, enabling seamless integration with Accessibility Agents and other tools.

  • Endpoints: /audit, /fix, /convert, /report (see documentation for details)
  • Automated unit tests using pytest and FastAPI’s TestClient ensure reliability and correctness
  • OpenAPI schema and full documentation included
  • Designed for workflow automation, batch processing, and integration with markdown-a11y-assistant and other Accessibility Agents

Learn more: MCP Server Documentation
