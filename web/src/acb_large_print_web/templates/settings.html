How this guided settings experience works

GLOW starts with standards-first defaults so every user gets a safe baseline. If your team has a repeat workflow, use this page to personalize defaults once, then GLOW automatically pre-fills forms each time.

Privacy first: nothing is saved unless you opt in. If you turn off Remember my settings, GLOW removes the saved local storage entry and returns to standard defaults.
