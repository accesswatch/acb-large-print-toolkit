About the ACB Large Print Guidelines

The American Council of the Blind (ACB) Large Print Guidelines specify formatting standards that make printed and digital documents readable for people with low vision. Key requirements include:

  • Arial font, 18-point minimum for body text
  • 22-point bold headings, 20-point bold subheadings
  • Flush left alignment (never justified)
  • Underline for emphasis (no italic, no bold in body text)
  • 1-inch margins on all sides
  • No hyphenation, no ALL CAPS formatting

About the APH Large Print Guidelines

The American Printing House (APH) provides complementary research-based large print guidance for educational and instructional content workflows.

For Excel and PowerPoint, the tool enforces Microsoft Accessibility Checker rules aligned with WCAG 2.2 AA: sheet names, table headers, alt text, reading order, slide titles, and more. Read the full guidelines reference.

External reference: APH publishes Guidelines for the Development of Documents in Large Print. APH alignment in this toolkit is fully integrated for the Release 1.2.0 submission workflow.
